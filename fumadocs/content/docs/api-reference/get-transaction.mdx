## `GET /api/v1/transactions/{transaction_id}`

Retrieve a single transaction including its items and payment methods.
Requires `X-API-Key` header.

## Path parameters

| Parameter | Description |
|-----------|-------------|
| `transaction_id` | The transaction ID (e.g., `TXN-REG-001`) |

## Example

```bash
curl http://localhost:8000/api/v1/transactions/TXN-REG-001 \
  -H "X-API-Key: SOLARA-SECRET-2026"
```

## Response `200 OK`

```json
{
  "data": {
    "id": "TXN-REG-001",
    "status": "CAPTURED",
    "total": "64.00",
    "subtotal": "50.00",
    "tax": "9.00",
    "shipping": "5.00",
    "currency": "USD",
    "is_cross_border": false,
    "items": [
      { "id": "ITEM-REG-001-A", "unit_price": "30.00", "quantity": 1 }
    ],
    "payments": [
      { "id": "PAY-REG-001", "type": "CARD", "amount": "64.00", "currency": "USD" }
    ]
  }
}
```

## Error responses

| Code | HTTP | Condition |
|------|------|-----------|
| `TRANSACTION_NOT_FOUND` | 404 | Transaction ID does not exist |
| `UNAUTHORIZED` | 401 | Missing or invalid API key |
