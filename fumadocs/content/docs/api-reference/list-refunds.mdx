## `GET /api/v1/refunds`

List all refunds. Requires `X-API-Key` header.

## Query parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `transaction_id` | string | Filter results to refunds for a specific transaction |

## Example

```bash
# All refunds
curl http://localhost:8000/api/v1/refunds \
  -H "X-API-Key: SOLARA-SECRET-2026"

# Filtered by transaction
curl "http://localhost:8000/api/v1/refunds?transaction_id=TXN-REG-001" \
  -H "X-API-Key: SOLARA-SECRET-2026"
```

## Response `200 OK`

```json
{
  "data": [
    { "refund_id": "RF-A1B2C3D4", "transaction_id": "TXN-REG-001", "..." }
  ],
  "meta": { "timestamp": "2026-02-25T10:00:00Z", "request_id": "..." }
}
```
